# Work Area

Put user-owned CAD work here.

- `models/`: durable CAD generators and generated artifacts.
- `briefs/`: modeling briefs, requirements, and notes.
- `references/`: user-provided drawings, photos, PDFs, and datasheets.
- `scratch/`: temporary experiments.

Avoid editing `..\upstream\text-to-cad` unless the task is explicitly about the upstream project.
